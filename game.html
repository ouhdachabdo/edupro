Fundamentals of Programming

1-4 days

* Basics :

-Learn a programming language commonly used in game development (EX : 'C#' , 'C++' , 'Python').

-Understand core programming concepts: variables, data types, control structures, loops, and functions.

-Explore object-oriented programming (OOP) concepts: classes, objects, inheritance, polymorphism, and encapsulation.

Game Development in Unity

1-20 days

* 2D Game Development :

-Learn about sprites, animations, and tilemaps.

-Explore 2D physics, collisions, and triggers.

-Understand camera control and parallax scrolling.

* 3D Game Development :

-Explore 3D modeling basics, importing assets, and textures.

-Learn about 3D physics, collisions, and raycasting.

-Understand lighting, shadows, and shaders.

* Scripting in Unity :

-Learn about player input and movement.

-Implement game mechanics: health systems, inventory, and AI.

-Understand object-oriented programming within Unity.

* Advanced Unity Features :

-Learn about particle systems and visual effects.

-Explore Unity's UI system for menus and HUDs.

-Implement multiplayer functionality with Unity's networking.

-Learn about optimization techniques for performance.

Graphics and Animation

1-10 days

* 2D Graphics :

-Learn about creating and animating sprites.

-Explore software like Photoshop, GIMP, or Aseprite for 2D asset creation.

* 3D Graphics :

-Learn about 3D modeling, rigging, and animation.
