Fundamentals of Graphic Design

1-10 days

¤ Design Principles :

* Color Theory :

-Understand color models (RGB, CMYK) and the color wheel.

-Learn about color harmony, contrast, and the psychology of color.

* Typography :

-Learn about typefaces, fonts, and their classifications.

-Understand the importance of hierarchy, spacing, and alignment in text.

-Explore kerning, leading, and tracking for better readability.

* Layout and Composition :

-Understand the rule of thirds, grids, and symmetry.

-Learn about balance, proximity, and alignment in layouts.

-Explore the importance of white space in design.

* Visual Hierarchy :

-Learn how to direct viewers' attention using size, contrast, and positioning.

-Understand the importance of focal points in design.

* Branding and Identity :

-Learn about creating logos, brand colors, and visual identity.

-Understand the importance of consistency in branding materials.

Adobe Photoshop

1-10 days

¤ Basics :

* Interface and Tools :

-Familiarize yourself with the Photoshop interface, panels, and tools.

-Learn about layers, layer masks, and blending modes.

-Explore basic tools like Move, Lasso, Magic Wand, and Pen Tool.

* Image Editing :

-Learn about cropping, resizing, and rotating images.

-Understand color correction techniques using Levels, Curves, and Color Balance.

-Explore retouching tools like Clone Stamp, Healing Brush, and Spot Healing.

* Working with Text :

-Learn about adding and formatting text in Photoshop.

-Explore text effects like shadows, gradients, and 3D text.

¤ Intermediate :

* Selections and Masks :

-Learn about advanced selection techniques using the Pen Tool and Select Subject.

-Understand how to create and refine masks for complex compositions.

* Image Manipulation :

-Explore advanced techniques like image warping, liquify, and puppet warp.

-Learn about creating composite images using multiple layers and masks.

* Filters and Effects :

-Understand how to apply filters (EX : 'Gaussian Blur' , 'Sharpen' ) and effects (EX : 'drop shadows' , 'bevels').

-Learn about smart filters for non-destructive editing.

¤ Advanced :

* Advanced Retouching :

-Explore high-end retouching techniques for portraits and product photography.

-Learn about frequency separation and dodging & burning.

* Photo Manipulation :

-Master techniques for creating surreal and imaginative images.

-Learn about advanced blending, perspective, and lighting effects.

* Automating Tasks :

-Explore actions and batch processing for automating repetitive tasks.

-Learn about using scripts to enhance productivity.

* 3D Design :

-Learn about creating and manipulating 3D objects in Photoshop.

-Understand how to integrate 3D elements into 2D designs.

¤ Projects and Practice :

* Basic Projects :

-Create a photo collage.

-Design a social media post.

-Edit and retouch a portrait photo.

* Intermediate Projects :

-Design a movie poster.

-Create a surreal photo manipulation.

-Develop a brand’s visual identity.

* Advanced Projects :

-Design a magazine cover.

-Create a complex photo composite.

-Develop a 3D product mockup.

Adobe Illustrator

1-10 days

¤ Basics :

* Interface and Tools :

-Familiarize yourself with the Illustrator interface, panels, and tools.

-Learn about vector graphics and the difference between vector and raster images.

-Explore basic tools like Pen Tool, Shape Tools, and Selection Tools.

* Working with Paths :

-Understand how to create and edit paths using the Pen Tool.

-Learn about anchor points, handles, and Bezier curves.

-Explore shape-building techniques using Pathfinder and Shape Builder.

* Colors and Gradients :

-Learn about applying and editing fills, strokes, and gradients.

-Explore color palettes and swatches for consistent branding.

¤ Intermediate :

* Typography and Text Effects :

-Understand how to create and manipulate text in Illustrator.

-Learn about creating custom type designs and text on a path.

-Explore advanced text effects like 3D text, shadows, and outlines.

* Illustration Techniques:

-Learn about creating custom illustrations using the Pen Tool and Shape Tools.

-Explore brush tools for hand-drawn effects and textures.

-Understand how to create and apply patterns and textures.

* Logo Design :

-Learn about designing logos and brand marks.

-Understand how to create scalable vector logos for various applications.

¤ Advanced :

* Advanced Vector Techniques :

-Explore the creation of complex vector illustrations and infographics.

-Learn about advanced path editing and vector masking techniques.

-Understand how to work with gradients, meshes, and blends for realistic effects.

* 3D Design :

-Learn about creating and manipulating 3D objects and text in Illustrator.

-Explore isometric design and 3D perspectives.

* Print and Production :

-Understand the principles of designing for print (EX : 'bleed' , 'CMYK' , 'DPI').

-Learn about preparing files for production (EX : 'print' , 'web' , 'packaging' ).

¤ Projects and Practice :

* Basic Projects :

-Design a vector icon set.

-Create a simple logo.

-Develop a business card.

* Intermediate Projects :

-Design a vector illustration.

-Create a branded stationery set.

-Develop a vector infographic.

* Advanced Projects :

-Design a complex logo with multiple variations.

-Create a detailed vector artwork.

-Develop a complete brand identity package.
